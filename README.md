# PM Certification Quiz

![Version](https://img.shields.io/badge/version-v1.3.22-blue)

**PM Certification Quiz** is a free, open-source Progressive Web App (PWA) for project management certification exam preparation. It provides practice quizzes covering all 8 PMP performance domains with offline support, timed questions, and detailed explanations.

## Features

*   **Comprehensive Coverage:** Questions from all 8 performance domains.
*   **Offline Capable:** Works without internet access after initial load (PWA).
*   **Timed Practice:** Simulates exam conditions with countdown timers.
*   **Instant Feedback:** Immediate correct/incorrect indication with explanations.
*   **Review Mode:** Analyze your answers at the end of the quiz.
*   **Privacy Focused:** No tracking, no cookies, all data stays on your device.
*   **Dark Mode:** Built-in theme toggling for comfortable reading.
*   **Mobile Optimized:** Designed for touch devices.

## Usage

1.  **Select a Question Bank:** Choose a specific domain from the dropdown menu.
2.  **Upload Your Own:** You can upload custom question files in JSON format.
3.  **Take the Quiz:** Answer questions before the timer runs out.
4.  **Review:** Check your score and review detailed explanations for every answer.
